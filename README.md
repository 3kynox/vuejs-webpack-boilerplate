# VueJs Boilerplate - Babel / Webpack / Sass

[WIP] My boilerplate for my vue.js projects.

#### TECHNOLOGIES

* VueJs
* GSAP - TweenMax
* Babel | ES2015
* Webpack
* ESLint
* ESDoc
* SASS

##### Install dependancies :
```shell
  npm i
```

##### Launch the project :
```shell
  npm start
```

The project will be launched at http://localhost:3000/

#### Build for production :
```shell
  npm run build
```

Hope you like it ¯\_(ツ)_/¯
